Evaluate the plan's todo list against the following criteria:

- **Consistency**: Each item must map directly to a step described in the plan — no gaps, no extra scope introduced.
- **Uniform granularity**: Items should be roughly equal in size and complexity; flag any that are disproportionately large or trivially small compared to the rest.
- **Clarity**: Each item should be phrased concisely so an AI agent can understand its scope and intent at a glance.
- **Actionability**: Each item should start with a verb and describe a concrete, executable action.
- **Dependency order**: Items must be ordered so that each item's dependencies appear before it. If a dependency cycle exists, split the involved items into smaller independent steps that can be ordered correctly.

Report any issues found, then propose a revised todo list if improvements are needed.
